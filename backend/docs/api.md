# 商品管理 API 说明

基础路径：`/api/v1`

## 1. 查询商品列表
- 方法：`GET`
- 路径：`/products`
- 响应示例：
```json
{
  "items": [
    {
      "id": 1,
      "name": "无线鼠标",
      "description": "静音按键，支持蓝牙与2.4G",
      "price": 89,
      "stock": 120
    }
  ]
}
```

## 2. 查询单个商品
- 方法：`GET`
- 路径：`/products/:id`
- 成功响应：`200`
- 失败响应：`404`（商品不存在）

## 3. 创建商品
- 方法：`POST`
- 路径：`/products`
- 请求体：
```json
{
  "name": "扩展坞",
  "description": "Type-C 扩展坞",
  "price": 199,
  "stock": 30
}
```
- 成功响应：`201`

## 4. 更新商品
- 方法：`PUT`
- 路径：`/products/:id`
- 请求体字段与创建接口一致
- 成功响应：`200`
- 失败响应：`404`（商品不存在）

## 5. 删除商品
- 方法：`DELETE`
- 路径：`/products/:id`
- 成功响应：`204`
- 失败响应：`404`（商品不存在）

## 6. 健康检查
- 方法：`GET`
- 路径：`/healthz`
- 成功响应：
```json
{
  "status": "ok"
}
```
